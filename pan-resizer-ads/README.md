## Installation

1. Upload the `pan-resizer-ads` folder to `/wp-content/plugins/`
2. Activate the plugin from WordPress admin panel
3. Go to **Appearance ‚Üí PAN Resizer ‚Üí Ads Manager**
4. Paste your ad codes in the respective input boxes
5. Click "Save All Ads"

## Ad Placement

### Social Bar
- **Position:** Left/Right side of website
- **Sticky:** Yes, stays visible while scrolling
- **Best for:** Engagement ads, share buttons

### Popunder
- **Position:** Background popup
- **Trigger:** User interaction based
- **Best for:** Conversions, offers

### Banner Ads
- **Position:** Between content sections
- **Format:** Horizontal banners
- **Best for:** Display ads, promotions
- **Multiple:** Can add up to 3 different banners

### Native Banner
- **Position:** Integrated within content
- **Style:** Matches site design
- **Best for:** Non-intrusive ads

### Smart Link
- **Position:** "Online PAN Resizer" header text
- **Format:** Hyperlink wrapper
- **Best for:** Affiliate links, tracking

## How to Get Ad Codes

Visit AdStyle website and create campaigns:
1. Log in to your AdStyle account
2. Create an ad campaign (Social Bar, Popunder, Banner, etc.)
3. Copy the HTML/JavaScript code
4. Paste it in the respective field in the Ads Manager
5. Save and you're done!

## Ad Code Format

Ad codes can be:
- HTML snippets
- JavaScript code
- Embed code from AdStyle
- Custom HTML with tracking

The plugin will automatically inject these codes in the right places on your website.
